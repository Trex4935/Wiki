# T-Rex 4935 RedBooks #

## What is a RedBook ##
A RedBook is a collection of dos & don't that we have learned for building robots and running the team since we were founded in 2013.  It is designed to distill lessons learned into simple bullet points that can quickly be absorbed and applied.
